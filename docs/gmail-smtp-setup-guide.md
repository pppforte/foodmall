# Gmail SMTP 설정 가이드

창업 문의가 접수되면 이메일로 알림을 받기 위해 Gmail SMTP 설정이 필요합니다.

---

## 사전 준비

- Gmail 계정 (Google Workspace 또는 개인 Gmail 모두 가능)
- 2단계 인증이 활성화된 계정

---

## 설정 순서

### 1단계: Google 2단계 인증 활성화

1. https://myaccount.google.com/security 접속
2. "Google에 로그인하는 방법" 섹션에서 **"2단계 인증"** 클릭
3. 안내에 따라 2단계 인증을 **활성화**

> 이미 2단계 인증이 켜져 있다면 이 단계는 건너뛰세요.

### 2단계: 앱 비밀번호 생성

1. https://myaccount.google.com/apppasswords 접속
2. **앱 이름** 입력란에 `프랜차이즈 홈페이지` 또는 원하는 이름 입력
3. **"만들기"** 버튼 클릭
4. 화면에 표시되는 **16자리 비밀번호**를 복사
   - 예시: `abcd efgh ijkl mnop`
   - 공백은 무시하고 `abcdefghijklmnop` 으로 입력해도 됩니다

> ⚠️ 이 비밀번호는 한 번만 표시됩니다. 반드시 복사해두세요.

### 3단계: 관리자 페이지에서 설정

1. 사이트 관리자 페이지 접속 (예: `https://도메인/admin`)
2. 로그인 후 **"사이트 설정"** 메뉴 클릭
3. **이메일 (SMTP) 설정** 섹션에서 다음 값 입력:

| 항목 | 입력값 |
|------|--------|
| **SMTP 호스트** | `smtp.gmail.com` |
| **SMTP 포트** | `587` |
| **SMTP 사용자** | Gmail 주소 (예: `myemail@gmail.com`) |
| **SMTP 비밀번호** | 2단계에서 복사한 16자리 앱 비밀번호 |
| **문의 수신 이메일** | 문의 알림을 받을 이메일 주소 |
| **발신자 표시** | 예: `프랜차이즈 문의 <noreply@example.com>` |

4. **"저장"** 버튼 클릭

---

## 테스트 방법

1. 사이트 메인 페이지에서 **창업 상담 문의** 폼 작성 후 제출
2. 설정한 **문의 수신 이메일**로 알림 메일이 오는지 확인
3. 관리자 페이지 **문의 관리**에서도 접수 내역 확인 가능

---

## 문제 해결

### "인증 실패" 오류가 발생하는 경우
- 2단계 인증이 정상적으로 활성화되어 있는지 확인
- **앱 비밀번호**를 사용했는지 확인 (Gmail 로그인 비밀번호가 아닙니다)
- 앱 비밀번호를 새로 생성해서 다시 시도

### "연결 시간 초과" 오류가 발생하는 경우
- SMTP 호스트가 `smtp.gmail.com`인지 확인
- SMTP 포트가 `587`인지 확인

### Google Workspace (회사 메일) 사용 시
- 관리자가 "보안 수준이 낮은 앱"을 허용했는지 확인
- 또는 동일하게 앱 비밀번호 방식 사용 가능

---

## 참고

- 앱 비밀번호는 언제든 https://myaccount.google.com/apppasswords 에서 삭제/재생성 가능
- Gmail 일일 발송 한도: 개인 계정 500통, Workspace 2,000통
- 문의가 접수되면 DB에도 저장되므로, 이메일 발송이 실패해도 관리자 페이지에서 확인 가능
